# 🚀 全网短视频/图文无水印解析下载系统

本系统是一个基于 **Spring Boot** 开发的高性能去水印集成方案，通过调用本地部署的解析引擎，实现对国内外主流社交平台的无水印资源抓取与流式转发下载。

## 🌟 项目简介

这是一个旨在解决“去水印”最后 100 米体验的集成项目。通过后端流转发技术，彻底解决了前端直接下载导致的文件损坏、跨域拦截等痛点，为自用和分享提供商业级的稳定体验。

### 核心功能：
- **全平台支持**：支持抖音（视频/图文）、Bilibili、小红书、TikTok 等多平台资源解析。
- **智能识别**：自动解析分享文案，精准识别并区分视频作品与图集作品。
- **多图瀑布流**：针对图文作品提供完整的图片预览瀑布流，并支持高清原图逐张下载。
- **流式转发下载**：由 Java 后端代理文件流，绕过浏览器跨域安全限制，确保下载文件 100% 完整。
- **零硬盘占用**：采用内存管道实时转发，视频数据不经过服务器硬盘落盘，即下即走，省空间更安全。

---

## 🏗️ 核心架构

本系统采用微服务中台化设计思路：
1. **解析大脑 (Analysis Engine)**：本地 Docker 运行，负责处理复杂的签名算法与风控 Cookie 管理。
2. **业务逻辑 (Backend)**：Spring Boot 开发，负责文案提取、数据清洗及跨域流转发。
3. **交互界面 (Frontend)**：极简渐变风格 UI，适配移动端与 PC 端。

## ⚠️ 核心依赖（部署前提）

本项目作为业务端，必须依赖底层的解析引擎才能正常运行。**在启动本项目前，请确保在服务器本地已通过 Docker 启动了以下服务：**

### 部署底层解析引擎
请部署大神的开源解析服务，并确保其运行在服务器的 `8000` 端口。

**推荐启动指令：**
```bash
docker run -d \
  --name douyin-api \
  --restart always \
  -p 8000:80 \
  -v /home/ubuntu/config.yaml:/app/crawlers/douyin/web/config.yaml \
  evil0ctal/douyin_tiktok_download_api
```
*提示：请务必按照该项目的要求在 `config.yaml` 中配置有效的 Cookie 以确保解析成功率。*
